El desarrollo de Git fue iniciado por Linus Torvalds en abril de 2005 después que la licencia gratuita de BitKeeper, el sistema propietario de control de versiones usado en el desarrollo del kernel de Linux desde 2002, fue revocada para Linux.


Se utilizaba para el desarrollo del kernel de Linux desde 2002. Git fue diseñado como un sistema de control de versiones distribuido, de código abierto, para satisfacer las necesidades del kernel de Linux y rápidamente se convirtió en el estándar para el desarrollo
